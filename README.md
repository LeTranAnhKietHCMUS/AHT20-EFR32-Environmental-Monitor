The project develops an environmental temperature and humidity monitoring system using the AHT20 sensor communicating via I2C with the EFR32xG21 microcontroller (Silicon Labs). The measured data is processed and distributed through three parallel channels: direct display on a 128×128 LCD screen, transmission to a PC via UART (115200 baud) for storage in CSV and SQLite databases, and wireless broadcasting via BLE to mobile devices. The system supports two operating modes — Auto-Logging and Command Mode — allowing users to flexibly control the sensor reading interval and BLE advertising frequency through UART commands. Experimental results demonstrate that the system operates stably with minimal measurement error, fully meeting the defined requirements.

Đồ án xây dựng hệ thống giám sát nhiệt độ và độ ẩm môi trường sử dụng cảm biến AHT20 giao tiếp I2C với vi điều khiển EFR32xG21 (Silicon Labs). Dữ liệu đo được xử lý và phân phối qua ba kênh song song: hiển thị trực tiếp lên màn hình LCD 128×128, truyền đến PC qua UART (115200 baud) để lưu vào cơ sở dữ liệu CSV và SQLite, và quảng bá không dây qua BLE đến thiết bị di động. Hệ thống hỗ trợ hai chế độ hoạt động là Auto-Logging và Command Mode, cho phép người dùng điều khiển linh hoạt chu kỳ đọc cảm biến và tần suất quảng bá BLE thông qua các lệnh UART. Kết quả thử nghiệm cho thấy hệ thống hoạt động ổn định với sai số đo lường nhỏ, đáp ứng đầy đủ các yêu cầu đặt ra.
